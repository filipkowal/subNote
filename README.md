# subNote

An application for writing down ideas, keeping a dairy or writing down things to remember (like movies to watch, books to read).
Based on concept of seemles, simple user experience.

It treats paragraphs as separate notes that can include subnotes.
New note is created when user starts new paragraph.
Subnotes can be hidden so only the main content is visible.

subNote stores data in users browser local storage so it can be accessed after closing the browser.
It connects to github server to obtain data about user.
